# 模式：training —— 課程 / 證照評估

候選人每問一門課或證照，就依 6 個維度評估：

| 維度 | 評估什麼 |
|------|----------|
| North Star 對齊 | 離目標更近還是更遠？ |
| Recruiter 訊號 | HM 看到這個出現在 CV 上會怎麼想？ |
| 時間與精力 | 週數 × 每週時數 |
| 機會成本 | 這段時間不能做什麼？ |
| 風險 | 內容是否過時？品牌是否薄弱？太過入門？ |
| Portfolio 產出 | 會不會產出可展示的 artifact？ |

## 裁決

- **做** → 4–12 週計畫，每週交付物 + scoreboard
- **不做** → 更好的替代方案 + 理由
- **限時做（最多 X 週）** → 精簡計畫，只做核心

## 優先順序

會提升「production-grade AI」可信度的訓練優先：
1. LLM evals 與測試
2. Observability 與 monitoring
3. Cost / reliability trade-off
4. AI governance 與 safety
5. Enterprise AI 架構
